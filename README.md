# GLA Frontend

## Pre-requisites
* [Node.js](https://nodejs.org/)

## Prepare
* `npm install`
* `npm install -g gulp` 
* `npm install -g bower`
* add `sudo` before the above install commands in Linux/MacOS

## Run
* `bower install`
* `gulp serve`


## Help
To clean up the current installation  

For bower  

* `bower cache list`
* `bower cache clean`
* `bower install`

For npm  

* `npm cache clean`
* `rm -rf node_modules/` or `rmdir node_modules` in windows
* `npm install`
